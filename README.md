# hydrogen-lottery-miniprogram
知晓云抽奖小程序

## 开发环境

```console
webpack: v3.8.1
nodejs : v6.9.2
```

## 安装依赖

```console
yarn install
```

## 开发实时编译

```console
yarn run dev        // 访问测试表数据
yarn run dev-prod   // 访问线上表数据
```

## build

```console
yarn run build
```
